# videogamesaustria
Tabletop used to access Google Spreadsheet for HTML

# Build Requirements
 * node.js -> https://nodejs.org/

## Install Build Environment
 * run: `npm install`
 * run: `npm install gulp -g` (optional)
 * run: `npm install bower -g` (optional)
 * run: `bower install` 
 
## Build
 * run: `gulp`
